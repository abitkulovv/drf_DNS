REST API интернет-магазина

Полноценный backend интернет-магазина на Django REST Framework с ролевой моделью, корзиной, заказами и системой отзывов. Покупатели совершают покупки, модераторы управляют товарами и контролируют качество отзывов. 


Возможности:
Покупатель(

регистрация и аутентификация

управление профилем и адресами доставки

добавление товаров в корзину

оформление заказов

публикация отзывов о товарах)

Модератор

(создание и редактирование товаров

модерация отзывов

обработка заказов)



Стек технологий

Python, Django, Django REST Framework

PostgreSQL

Docker + Docker Compose

JWT-аутентификация
